# 💰 Smart Expense Analyzer & Financial Advisor

A full-stack **MERN** personal finance app with real-time analytics, budget planning, and bill reminders — built with a clean white/light theme.

---



## 🚀 Features

| Feature | Details |
|---|---|
| ✅ Authentication | Register / Login with JWT tokens |
| ✅ Transactions | Add income/expenses with category, date, payment method |
| ✅ Analytics | Category-wise charts using Recharts |
| ✅ Budget Planner | Set monthly budgets per category with progress bars |
| ✅ Bill Reminders | Track recurring bills, mark as paid, overdue alerts |
| ✅ Reports | Donut chart, area chart, bar chart, category table |
| ✅ Multi-user | Every user has isolated data via JWT |
| ✅ Light Theme | Clean white/slate Tailwind CSS design |

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Database | MongoDB + Mongoose |
| Backend | Node.js + Express.js |
| Auth | JWT + bcryptjs |
| Frontend | React 18 + Vite |
| Styling | Tailwind CSS 3 |
| Charts | Recharts |
| HTTP Client | Axios |
| Notifications | react-hot-toast |
| Icons | lucide-react |
